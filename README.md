# spam-ham-feedback-app
A Streamlit demo for Email/Message Spam Detection with feedback and charts.
Built with scikit-learn, Streamlit, matplotlib, and seaborn. 
Model: TF-IDF vectorizer + Multinomial Naive Bayes
